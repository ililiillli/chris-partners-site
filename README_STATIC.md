# JS&PARTNERS Static Website

This is a production-ready static version of the JS&PARTNERS website, built with HTML, CSS (Tailwind), and JavaScript.

## Files
- `index.html`: The main structure of the website.
- `style.css`: Custom styles and component definitions.

## Features
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.
- **Animations**: Uses AOS (Animate On Scroll) for smooth entry effects.
- **Modern UI**: Centered Hero section, refined card designs, and a sophisticated color palette.
- **Interactive**: Smooth scrolling for anchor links and a functional mobile menu.

## How to Use
1. Download `index.html` and `style.css`.
2. Place them in the same directory.
3. Open `index.html` in any web browser.

## Customization
- **Colors**: You can modify the `tailwind.config` in `index.html` or the color variables in `style.css`.
- **Images**: Replace the Unsplash URLs with your own local image paths if needed.
- **Content**: Edit the text directly in `index.html`.
